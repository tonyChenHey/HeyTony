### 基本语法

>如果实在忘记了一些语法，用html格式也是可以的噢

#### 标题
用 ***#*** 表示需要写的标题，打多少个表示多少级标题，比如一个表示 **h1** ，两个表示 **h2**，举个例子（三级标题）：  
```
    ### 三级标题
```

#### 换行
可以使用 `<br>` 来表示换行，或者以**两个**或**多个空格**结束后敲**Enter** 来表示换行

#### 着重
在文字的前后分别加上`**`就可以了，这么写:
```
    为了赚到**钱**，他放下了**面子**
```
